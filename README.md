# திருக்குறள் Daily

A daily Thirukkural PWA. One kural a day, every day.

## Live
https://vignesh-selva.github.io/thirukkural/

## Stack
- Vanilla HTML/JS — no framework
- Kurals from `kurals.json` (bundled, no API dependency)
- Hosted on GitHub Pages (free)

## Features
- Daily kural based on day of year
- Tamil original + English verse + meaning
- Browse all 1330 kurals with ‹ › navigation
- Copy / native share
- Installable PWA (Add to Home Screen)
- Offline support via service worker

## Files
- `index.html` — the entire app
- `kurals.json` — all 1330 kurals
- `manifest.json` — PWA config
- `sw.js` — service worker
- `icon.svg` — app icon
